<p>Selenium 是一款自动化测试工具，能够模拟人工对浏览器的操作。而现在WEB技术发展的非常快，其中<a href="https://zh.wikipedia.org/wiki/AJAX">AJax</a>在网页中的使用非常能够普遍，但是Selenium
能够很好地加载被AJax渲染的网页，所以对于爬虫工程师掌握Selenium是非常有必要的。<p>
<h2>安装</h2>
<ul>
    <li>pip install selenium</li>
    <li>根据浏览器和其版本，下载相应的驱动：<a href="https://chromedriver.chromium.org/">Chrome</a>，下载好之后，点击安装，路径要添加到Path中，要不然程序识别不出来</li>
</ul>
